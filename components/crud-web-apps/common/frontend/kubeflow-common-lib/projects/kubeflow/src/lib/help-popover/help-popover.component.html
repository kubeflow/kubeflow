help

Create a filter query by entering either a property name or just a value to narrow your results.

Example: Name: filterValue

Filter empty values by entering double quotes: Name: ""

Status

Filter based on the objects' status, by using:

  • The value shown in the tooltip
  • The status types (i.e. ready, waiting, warning, error, unavailable, uninitialized, terminating, stopped)

Date

Filter timestamp values based on:

  • The "-" character for empty date values: Created at: -
  • The relative time text, shown in the column (i.e. X months ago)
  • The substring of the timestamp, that is shown in the UTC part of the tooltip
